Alle PDDL-Dateien wurden in Visual Studio Code in Kombination mit dem PDDL-Plugin angefertigt.
Der Ordner kommt mit einem lokalen enhsp-solver und einer Einstellungsdatei in .vscode, in der dieser Solver automatisch ausgewählt sein sollte.
Weiterhin sollte automatisch diese Option "-s gbfs -h hmax" gewählt werden. Mit dieser Einstellung wurden alle Pläne erstellt.
Es sollte möglich sein ohne weitere Einstellungen jede Datei auszuwählen und abzuspielen. Falls nicht, muss genanntes manuell eingestellt werden.
